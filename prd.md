# 产品名称
- 图片字幕生成器

# 目标与价值
- 为任意图片快速添加多行字幕，并生成同款“底部每行同背景、分割切片感”的字幕效果图片
- 面向内容创作者、社媒运营，提高出图效率与一致性

# 关键特性
- 本地图片上传与预览
- 多行字幕输入
- 可调参数：字幕高度、字体大小、字体颜色、轮廓颜色、字体样式、字体粗细
- 底部多行字幕的统一矩形背景（同色同高），行间留缝形成切割感
- 导出生成图片、保存图片
- 生成成功提示

# 用户流程
1. 选择图片
2. 配置字幕样式参数
3. 输入多行字幕
4. 预览自动渲染
5. 点击生成字幕图片
6. 保存/下载

# 详细功能需求
1. **图片上传**
   - 支持拖拽/点击选择；支持 jpg/jpeg/png/webp；最大 20MB
   - 保留EXIF方向；显示文件名；错误提示（格式不支持/过大）
2. **字幕内容**
   - 多行文本框，按换行拆分为独立字幕行
   - 支持中英文及多语言；限制最多 12 行（可配置）
3. **样式设置**
   - 字幕高度（行背景矩形高度，默认 80px，可 40–200）
   - 字体大小（默认 40px，可 16–96）
   - 字体颜色（默认 #FFFFFF）
   - 轮廓颜色（默认 #000000，描边宽度自动随字体大小调整，比例 1.5–2.5px，可微调）
   - 字体样式（默认 微软雅黑；下拉常用无衬线/系统字体）
   - 字体粗细（默认 正常；可选 细/正常/中/粗）
   - **高级参数**（默认值）：
     - 背景颜色（#000000）
     - 背景不透明度（0.6）
     - 行间距（8px，控制切割缝隙）
     - 内边距（左右 24px）
     - 圆角（8px）
     - 字幕底部距离（24px）
     - 对齐方式（左/中/右，默认左）
4. **渲染规则（核心“切割感”）**
   - 将每一行字幕绘制在同色同高的背景矩形上
   - 所有行背景矩形宽度一致（与图片宽度相同或设定的安全区宽度）
   - 行与行之间保留固定缝隙（行间距），不合并为整体面板
   - 背景矩形彼此独立，形成“切片”视觉
   - 字体描边始终启用，提升可读性
   - 自动避免越界：若底部空间不足，整体字幕区上移
5. **预览**
   - 实时更新；等比居中显示；支持缩放预览 50%/100%
6. **生成与保存**
   - 一键生成，导出 PNG（默认）/JPEG（可选，高质量）
   - 下载文件命名：originalName_subtitled_yyyyMMddHHmmss
   - 成功提示：绿色条“字幕图片生成成功!”
7. **交互与校验**
   - 必填：已上传图片且至少一行字幕
   - 输入限长：每行最多 60 字符（超出提示换行或缩小字体）
   - 状态保持：刷新不丢失（本地存储最近一次设置）

# 非功能需求
- 浏览器兼容：Chrome/Edge/Firefox/Safari 最新两个大版本
- 性能：5000×5000 像素内 300ms 内完成渲染
- 安全：仅本地处理，无服务端上传
- 可访问性：键盘操作、标签关联、对比度达标

# 技术实现建议
- **前端**：Canvas 或 OffscreenCanvas；React + Vite + Tailwind CSS
- **文本描边**：ctx.strokeText + ctx.fillText，描边宽度与字体大小联动
- **背景矩形**：按行计算 y 坐标；统一高度 = 字幕高度；宽度 = 画布宽度；圆角路径
- **文本排版**：基线 middle；垂直居中到矩形；左右内边距控制文本起始 x
- **高清输出**：以原图分辨率渲染；预览缩放显示
